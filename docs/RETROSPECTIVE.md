## 2. 가장 오래 막혔던 구간

막힌 구간: React 프론트엔드와 FastAPI 백엔드를 연결하는 과정에서 개발 환경(Node.js, npm, CORS, Continue 설정)을 구축하는 단계

원인:  Node.js 환경 변수(PATH) 문제로 node, npm, npx 명령어가 실행되지 않음.
      React와 FastAPI를 연결하는 과정에서 CORS 오류가 발생함.
      Continue와 Gemini API 연동 과정에서 설정 방식이 수업 자료와 달라 혼란이 있었음.


해결 방법: Node.js 환경 변수(PATH)를 수정하여 npm을 정상적으로 사용할 수 있도록 설정
          FastAPI의 CORS 허용 주소를 React 개발 서버 포트에 맞게 추가
          Continue의 config.yaml에 Gemini API Key를 등록하여 AI 코딩 환경을 구축하고, 브라우저와 API 연동을 반복 테스트하며 문제를 해결함
